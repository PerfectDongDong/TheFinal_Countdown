# 生命倒计时 - The Final Countdown

一个跨平台的生命倒计时和任务管理应用程序，支持PC端桌面浮动窗口和移动端浏览器访问。

## 功能特性

### 1. 生命倒计时功能
- 输入生日信息（年/月/日）
- 设置预期寿命
- 实时显示剩余生命时间（年、月、日）
- 显示生命进度条

### 2. 任务管理功能
- 添加未完成任务
- 任务分类到指定年份
- 任务查看、编辑和删除
- 按年份筛选任务

### 3. 界面与主题
- 简洁直观的用户界面
- 响应式设计，适配不同屏幕尺寸
- 支持浅色/深色主题切换
- 跟随系统主题

### 4. 数据管理
- 本地数据存储（LocalStorage）
- 数据导出/导入功能
- 数据清除功能

### 5. 桌面浮动窗口
- 可拖动位置
- 可调整透明度
- 最小化/展开功能
- 快速访问完整版本

## 使用方法

### 完整版本
1. 在浏览器中打开 `index.html`
2. 在"倒计时"标签页设置您的出生日期和预期寿命
3. 点击"保存设置"开始倒计时
4. 切换到"任务管理"标签页添加和管理任务

### 浮动窗口版本（PC端）
1. 在浏览器中打开 `float-window.html`
2. 点击并拖动顶部标题栏可以移动窗口位置
3. 使用滑块调整窗口透明度
4. 点击"设置"按钮配置个人信息
5. 点击"完整版本"打开完整功能页面

## 文件说明

- `index.html` - 完整功能版本的主页面
- `float-window.html` - 桌面浮动窗口版本
- `styles.css` - 样式文件，包含主题和响应式设计
- `app.js` - 完整功能版本的JavaScript逻辑
- `package.json` - 项目配置文件

## 技术栈

- HTML5
- CSS3（CSS变量实现主题）
- Vanilla JavaScript（无框架）
- LocalStorage（本地数据存储）

## 跨平台兼容性

- **PC端**：支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- **移动端**：支持Android和iOS的现代浏览器
- **数据同步**：通过导出/导入JSON文件在不同设备间同步数据
